# Day 1 - Localhost Traffic Analysis & TCP Behavior Investigation

## Objective
To capture and analyze network traffic using Wireshark and understand TCP communication patterns.

## Analysis Summary
- Captured live network traffic
- Identified localhost communication (127.0.0.1)
- Analyzed TCP packet with [PSH, ACK] flags
- Determined traffic as internal system communication

## Security Assessment
- No suspicious external communication detected
- No abnormal packet patterns observed
- Traffic classified as normal system behavior

## Skills Demonstrated
- Packet analysis using Wireshark
- TCP flag interpretation
- Traffic classification (internal vs external)
- Basic SOC investigation workflow

## Tools Used
- Wireshark
